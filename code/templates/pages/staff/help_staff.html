Log In Help
Is your username and password correct?

To log into Blackboard use your UQ Sign In username and password. If you don't know your password contact the ITS Helpdesk on x56000, or by email help@its.uq.edu.au.

Have you checked that you are using a compatible browser?

Blackboard has a webpage specifying which browsers will perform optimally with Blackboard. It can be viewed here: Browsers supported by Blackboard

Software and Plugins

To use Blackboard your computer may need to have some software and plugins installed. If you are accessing Blackboard from a UQ student lab, then all of the necessary software should already be installed, but to access from home you may need to install some software. Check with AskUS if you need assistance.
